
🚀 Entry Level Job Requirement Analysis Dashboard
📌 Project Overview

This project analyzes more than 15,000 entry-level job listings across India to understand hiring trends, required skills, salary distribution, job locations, and experience requirements.

The dashboard helps students, job seekers, and recruiters explore the current entry-level job market through interactive visualizations.

📊 Dashboard Features
Overview of job market statistics
Company-wise job listings
Job location analysis
Salary analysis
Experience-level analysis
Technical skills demand
Soft skills demand
Skill combinations
Interactive filters

📈 Key Insights
Analyzed 15.1K+ Job Listings
3.4K+ Companies
49 Locations
SQL and Python are the most demanded technical skills.
Communication and Problem Solving are the most requested soft skills.
Salary trends vary significantly across experience levels.
